# 소상공인 실태조사 데이터 분석 프로젝트

## 프로젝트 배경
대한민국 자영업자(특히 영세 요식업)를 장기적으로 돕는 사업의 기반을 마련하기 위한 탐색 프로젝트.
이 분석의 목적은 "자영업자가 실제로 겪는 문제 중 내가 풀 수 있는 의미 있는 문제는 무엇인가"를 데이터로 탐색하는 것이다.

---

- **기준**: 2023년 12월 31일

## 분석 목표 (우선순위 순)

### 1. 타겟 세그먼트 규모 파악
- 숙박·음식점업 (`산업대분류코드 == 'I'`) 필터링
- 종사자 수 구간별 분포: 1명(대표자만), 2~3명, 4~9명, 10명 이상
- **핵심 타겟**: 종사자 1~3명 구간의 절대 수 및 비율

### 2. 비용구조 분석 (종사자 규모별 비교)
- 매출 대비 각 비용 항목 비율 계산
  - 원가율 = 매출원가 / 매출액
  - 인건비율 = 급여총액 / 매출액
  - 임차료율 = 임차료 / 매출액
  - 기타비용율 = 기타금액 / 매출액
  - 영업이익률 = 영업이익 / 매출액
- 종사자 1~3명 vs 4명 이상 비교
- 매출 구간별(예: ~5천만, 5천~1억, 1~2억, 2억 이상) 비용구조 분포

### 3. "잘 버티는 가게" vs "못 버티는 가게" 비교
- 영업이익률 상위 25% vs 하위 25% 집단 비교
- 어떤 비용 항목에서 차이가 나는가
- 디지털 도입 여부, 프랜차이즈 여부, 임차 여부 등 운영 특성 차이

### 4. 애로사항 분석
- 숙박·음식점업 전체에서 애로사항 빈도 순위
- 종사자 규모별 애로사항 차이
- 영업이익률 하위 집단의 주요 애로사항

### 5. 폐업 의향 분석
- 사업전환_운영계획코드 별 분포
- 폐업 의향 집단의 특성 (매출, 이익률, 애로사항)

## 대시보드 요구사항

**기술 스택**: Python (pandas)으로 데이터 전처리 → JSON으로 저장 → 단일 HTML 파일로 시각화
- 차트 라이브러리: Chart.js (CDN) 또는 Plotly.js (CDN)
- 외부 의존성 없이 HTML 파일 하나로 완결될 것
- Python 스크립트(`analysis.py`)가 JSON 데이터를 생성하고, `dashboard.html`이 이를 읽어 렌더링하는 구조
- 단, JSON을 별도 파일로 두지 않고 HTML 안에 인라인으로 삽입하여 파일 하나로 완결할 것

**구성 섹션**:
1. **개요**: 전체 vs 요식업 vs 요식업 소규모(1~3명) 규모 비교 카드
2. **비용구조**: 종사자 규모별 비용 항목 비율 비교 (스택 바 차트)
3. **수익성 분포**: 영업이익률 히스토그램 (규모별 색상 구분)
4. **애로사항 Top 5**: 요식업 소규모 기준 막대 차트
5. **생존 vs 폐업**: 운영계획 분포 및 폐업 의향 집단 특성

## 주의사항
- 매출액이 0이거나 극단적인 이상값(예: 매출액 > 100억)은 분석 전 필터링 또는 별도 처리
- 비율 계산 시 매출액이 0인 행 제외
- 가중치(`사업체수가중값`) 적용 여부를 항상 명시할 것
- 금액 단위는 만원 기준으로 통일, 차트에 단위 명기
- 코드값은 반드시 한글 레이블로 변환하여 출력
